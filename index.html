Big Mouse is a small UI library aimed to extend the mouse capabilities by bringing big data right into the mouse cursor with useful and beautiful visualizations.

Here's a simple demo to highlight the possibilities of Big Mouse. Just move your mouse cursor right to the latest blog posts panel on the left. That's a traditional blog widget panel in which you may discover the list of latest blog entries posted by title. Once Big Mouse is enabled, the mouse cursor will be giving you extra information for each post as described below.

  • Number of particles => Number of comments in post (more particles, more comments)
  • Radius of Particles => Post Reading time (the smaller, the faster to read)
  • Size of particles => Comments length (the bigger, the lengthy)

The attributes were merely selected for demonstration purposes. Other many attributes could be used as they make more sense as particles speed, color, brightness, distortion, etc. We need contributors, join us!
