BIG MOUSE

A small UI library aimed to extend the mouse capabilities by bringing big data right into the mouse cursor with useful and beautiful visualizations

Above's a simple demo to highlight the possibilities of Big Mouse. Just hover the links in the panel above with the mouse cursor. Imagine that's a traditional blog widget panel in which you may discover the list of latest blog entries posted by title. Once Big Mouse is enabled, the mouse cursor will be giving you extra information for each post in the panel as described below. Imagine for instance that particles in red are comments marked as spam.

Number of particles: Number of comments in post (more particles, more comments)

Color of particles: Amount of different users commenting the post (the more colors, the more users)

The attributes were merely selected for basic demonstration purposes. Other many attributes could be used to reference other interesting/valuable variables in your application domain as they make more sense as for instance particles size, speed, color, brightness, distortion, radius, etc. Particles system is actually a plugin, so you could interchange it with other puglins displaying different animations or interactive charts.
